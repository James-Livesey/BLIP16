All console hardware access is available under the BLIP16 object.

BLIP16.buffer is a Uint8Array which contains an item for each pixel on the display, where the first item of the array is the top-leftmost pixel, and the last item is the bottom-rightmost, where each item is sequenced in a left-to-right, then top-to-bottom order.

Each item in BLIP16.buffer is a number in the range of 0 to 15 inclusive, with each value representing a different colour, as indexed in the BLIP16.palette array. Each item in that array is a 24-bit colour value.

Buttons are represented as an object of boolean values under BLIP16.buttons. The available keys for this object are up, down, left, right, a and b. Their values are true when their respective button is pressed, and false when not.
